# Obtain Remote IP Address

All providers write `ansible_host` and `ansible_port` to the autogenerated
inventory when a VM is created. Query them with:

```shell
ansible-inventory --host <hostname>
```

Replace `<hostname>` with the hostname shown by `ansible-inventory --graph`.
The output includes `ansible_host` (IP address) and `ansible_port` for the
provisioned instance, regardless of provider.
